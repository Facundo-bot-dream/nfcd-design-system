Ink divider — the short "tick" echoes the rule under the wordmark; use it between an eyebrow and a title, or to punctuate sections.

```jsx
<Rule />                    {/* short ink tick */}
<Rule variant="full" />     {/* full hairline */}
<Rule variant="tick" tone="wine" />
```

Variants: `tick` (48px, default), `full` (hairline), `strong` (2px). Set `vertical` for an inline separator; `tone`: `ink` · `wine` · `faint`.
